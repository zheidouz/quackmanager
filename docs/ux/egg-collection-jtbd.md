# Job-to-be-Done: Daily Egg Collection

## Job Statement
> When I'm doing my morning farm rounds, I want to quickly record how many eggs each duck house produced, so I can track daily production trends, know when to sell, and spot health issues early.

---

## User Persona

| Attribute | Description |
|-----------|-------------|
| **Role** | Duck farm owner-operator (primary) / farm worker (secondary) |
| **Skill level** | Low tech literacy — prefers simple interfaces, minimal typing |
| **Primary device** | Mobile phone (360–428px screen), often with dirty/wet hands |
| **Context** | Outdoors, early morning, often in a hurry, may have poor connectivity |
| **Frequency** | Daily (every morning without fail) |
| **Tech comfort** | Comfortable with basic phone apps, not with complex software |
| **Accessibility** | May wear gloves, may have reduced dexterity in cold weather |

---

## Current Solution & Pain Points

| Aspect | Current | Pain Point |
|--------|---------|------------|
| **Recording** | Paper notebook + pen | Gets wet, pages lost, handwriting illegible |
| **Calculation** | Mental math or calculator | Easy to make arithmetic errors |
| **Storage** | Notebook on a shelf | No backup, no trend analysis |
| **Sharing** | Verbal or SMS to farm owner | Information gets lost |
| **Analysis** | None — just raw counts | Can't spot trends or issues early |
| **Time cost** | ~5 minutes to find notebook, write, calculate | Wastes time every single day |

**Consequence of failure**: Missed production trends → not noticing when egg output drops → delayed response to duck health issues → revenue loss.

---

## Key User Needs

1. **Open app, tap, done** — Record eggs in under 10 seconds with one hand
2. **No typing** — Use number steppers, pre-filled dates, and large buttons
3. **Works offline** — No internet in the duck house
4. **Shows today's status** — "Have I recorded eggs today?" at a glance
5. **Prevents double-entry** — Can't accidentally record the same day twice
6. **Quick correction** — Easy to fix if wrong number was entered

---

## Success Metrics

| Metric | Target |
|--------|--------|
| Time to record daily count | < 10 seconds |
| Completions without errors | > 99% |
| Daily completion rate | 100% of farm days recorded |
| Offline reliability | Works without internet 100% of the time |
| User satisfaction | "Easier than paper" |
